# TODO LIST APP
This application is a simple todo list developed in Go.

## ディレクトリ構成
```
/todo-app
|
|-- /app
|   |-- /handlers

|
|-- /resources
|   |-- /view
|       |-- /todos
|       |   |-- index.html
|       |
|       |-- layout.html
|
|-- main.go
|-- go.mod
|-- go.sum
|-- README.md
```
